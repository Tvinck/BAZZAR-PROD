# Connect Mobile — UI Kit

Interactive recreation of the employee app (iOS, dark). Open `index.html`: 390×844 phone frame with working tab bar, project switcher, employee chat (send works), notifications overlay.

Screens: HomeScreen, ServicesHRScreen, ChatScreen, MoreScreen, NotificationsScreen, UsefulServicesScreen — all compose the primitives from `components/`; no styles are re-implemented here.

Per the product brief this kit intentionally omits: one-time password service, sound/mic check, biometric login toggle, licenses row.
