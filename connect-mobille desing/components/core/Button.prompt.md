Button — yellow `primary` for the one main CTA per screen; `tonal` (dark fill, blue text) for secondary in-card actions; `plain` for text actions; `blue` for send/confirm.

```jsx
<Button variant="primary" block>Начать разметку</Button>
<Button variant="tonal" block>Начать работу</Button>
```

Press state: slight scale-down + opacity. In-card CTAs use `block`.
