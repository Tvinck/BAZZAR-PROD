# 📚 Документация Bazzar Pixel

Добро пожаловать в архив документации проекта Bazzar Pixel. Здесь собраны все инструкции, схемы базы данных и планы развития.

## 📂 Структура

### 🗄️ [Database](./database/)
- `supabase_schema.sql` — Актуальная схема базы данных.
- `ORDERS_AND_TRIGGERS.sql` — Логика обработки заказов и триггеры.
- `PAYMENT_RPC.sql` — Функции для работы с платежами через RPC.
- `SQL_TO_RUN_IN_SUPABASE.sql` — Набор SQL-запросов для быстрой настройки.
- `RETENTION_POLICY_7_DAYS.sql` — Политика удержания данных.

### 🗺️ [Planning](./planning/)
- `ROADMAP.md` — Генеральный план развития проекта.
- `TODO.md` — Список текущих задач и исправлений.
- `NEXT_STEPS.md` — Ближайшие приоритеты.
- `PROJECT_IDEAS_ROADMAP.md` — Идеи для будущих фич.

### 📖 [Guides](./guides/)
- `QUICKSTART.md` — Быстрый старт для разработчиков.
- `LOCAL_TESTING.md` — Инструкция по локальному тестированию.
- `NGROK_SETUP.md` — Настройка туннеля для вебхуков.
- `LAUNCH_MANUAL_RU.md` — Руководство по запуску.

### 🛠️ [API](./api/)
- `API_REFERENCE.md` — Описание основных эндпоинтов бэкенда.

### 🏛️ [Archive](./archive/)
- Старые отчеты, результаты завершенных этапов и исторические документы.

---

## 🚀 Основные файлы в корне
- `README.md` — Описание проекта и инструкции по развертыванию.
- `package.json` — Зависимости и скрипты запуска.
- `vercel.json` — Конфигурация для деплоя на Vercel.
- `server/` — Исходный код бэкенда.
- `src/` — Исходный код фронтенда.
