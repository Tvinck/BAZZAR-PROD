# 📄 ФУНКЦИОНАЛЬНЫЙ ОТЧЕТ: BAZZAR PIXEL

Этот документ описывает архитектуру, основные механики и функции проекта Bazzar Pixel.

## 1. Обзор Системы
Bazzar Pixel — это многофункциональный AI-бот и веб-приложение для генерации контента (изображения, видео, стикеры) с интеграцией Telegram и системой экспертных чатов.

### Технологический стек:
- **Frontend:** React, Vite, Framer Motion, Telegram Mini Apps SDK.
- **Backend:** Node.js (Express), Supabase (DB & Auth & Storage), pg-boss (очереди).
- **AI Интеграции:** Kie.ai (основной провайдер), Replicate, ElevenLabs, Yandex.

---

## 2. Основные Механики

### 🔐 Авторизация и Роли
- **Telegram Auth:** Основной метод входа через Telegram Web App.
- **Role-Based Access:** Система ролей (user, admin). Админ-панель защищена проверкой роли.
- **Web Login:** Возможность входа через OAuth (Yandex, VK) для веб-версии.

### 🎨 AI Генерация
- **Text-to-Image:** Генерация изображений по текстовому запросу.
- **Video Greetings:** Создание видео-поздравлений от "звезд" (синхронизация губ/лиц).
- **Stickers:** Создание стикер-паков из сгенерированных изображений.
- **Magic Tools:** 
  - Magic Eraser (Inpainting)
  - Face Swap
  - Avatar Trainer (LORA тренировка)
- **Prompt Enhancement:** Улучшение пользовательских запросов через AI перед генерацией.
- **Кэширование:** Повторные запросы на идентичные промпты выдаются мгновенно из кэша.

### 💬 Чат с Экспертами
- **Персонажи:** Список специализированных AI-ассистентов (Психолог, Финансист, Доктор, Жириновский и др.).
- **Механика:** Каждый эксперт имеет свой `systemPrompt`, определяющий стиль и знания.
- **Лимиты:** Система бесплатных сообщений и платная подписка/кредиты.

### 💰 Экономика и Платежи
- **Credits (Pixel Coins):** Внутренняя валюта для оплаты генераций.
- **Transactions:** Полная история списаний и пополнений.
- **XP & Levels:** Система опыта за активность, влияющая на статус пользователя.
- **Transactions UI:** Интеграция с Telegram Stars и внешними шлюзами.

---

## 3. Блоки и Структура Кода

### Backend (`/server`)
- `/routes`: API эндпоинты (админка, генерация, чат, пользователи).
- `/middleware`: Проверка авторизации и прав доступа.
- `/services`: Логика взаимодействия с внешними AI API и платежками.
- `/bot`: Обработчики команд Telegram бота (уведомления, рассылки).
- `/queue`: Система очередей для тяжелых задач.

### Frontend (`/src`)
- `/views`: Основные экраны (Home, Gallery, History, Profile).
- `/context`: Глобальное состояние (User, Theme, Sound, Toast).
- `/components`: Переиспользуемые UI блоки и модальные окна.
- `/hooks`: Кастомная логика для Telegram и маркетинга.
