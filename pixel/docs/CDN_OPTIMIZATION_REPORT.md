# Отчёт об оптимизации ассетов

## До оптимизации:
- public: 34MB (public/images: 17MB, public/videos: 11MB)

## После оптимизации:
- Локально осталось: 13 MB
- Вынесено в CDN: Все тяжелые файлы > 100KB, а также webp конвертации.
- Экономия: ~61% локального веса проекта.

## Файлы в CDN:
- Все изображения и видео загружены на бакет `assets` Supabase Storage.
- Скрипты для оптимизации добавлены в `package.json`
- `useCDN` хук успешно забирает файлы из CDN вместо локальной папки.
